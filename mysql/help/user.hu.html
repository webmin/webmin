Minden felhasználóra az alábbi paraméterek alkalmazhatók :
Felhasználónév
Ebbe a mezőbe beírhat egy tetszőleges felhasználónevet, de választhatja a Névtelen felhasználó opciót is, mely esetben a jogosultságok minden olyan felhasználóra vonatkoznak, akik nem szerepelnek a Felhasználói jogosultságok listáján.

Jelszó
A felhasználó által bejelentkezéskor beírandó jelszó. Ha Üres, úgy nincs szükség jelszóra.

Gazdák
Ama gazdagép IP címe, vagy gazdaneve, melyről a felhasználó bejelentkezhet. Ez a mező hagyományos SQL kifejezést is tartalmazhat, melynek segítségével több gazda is definiálható, például 192.168.1.% vagy %.foo.com.

Jogosultságok
Ama MySQL műveletek, amelyeket a felhasználó bármely adatbázison végrehajthat.
