Minden felhasználóra az alábbi paraméterek alkalmazhatók :
Felhasználónév
Ebbe a mezõbe beírhat egy tetszõleges felhasználónevet, de választhatja a Névtelen felhasználó opciót is, mely esetben a jogosultságok minden olyan felhasználóra vonatkoznak, akik nem szerepelnek a Felhasználói jogosultságok listáján.

Jelszó
A felhasználó által bejelentkezéskor beírandó jelszó. Ha Üres, úgy nincs szükség jelszóra.

Gazdák
Ama gazdagép IP címe, vagy gazdaneve, melyrõl a felhasználó bejelentkezhet. Ez a mezõ hagyományos SQL kifejezést is tartalmazhat, melynek segítségével több gazda is definiálható, például 192.168.1.% vagy %.foo.com.

Jogosultságok
Ama MySQL mûveletek, amelyeket a felhasználó bármely adatbázison végrehajthat.
