MySQL Adatbázis Szerver
A MySQL egy egyszerű adatbázis szerver Unix rendszerekhez. Többféle adatbázist, fejlett hozzáférési szabályokat, több adattípust, illetve az SQL szintaxis nagy részét támogatja. A szerver által kezelt adatbázisok mindegyike több táblázatot tartalmazhat, a táblázatokon belül több adatsorral.

A főoldal tetején az adatbázis-ikonok listája látható, alatta az új adatbázis hozzáadásához szükséges hivatkozással. Minden MySQL szerver tartalmazza minimálisan a mysql adatbázist, amely a konfigurációs és hozzáférés-vezérlési táblázatokat tartalmazza. Az adatbázisok alatt a hozzáférés-vezérléshez szükséges ikonok találhatók, míg a lap alján a szerver leállításához és elindításához szükséges gomb található.
