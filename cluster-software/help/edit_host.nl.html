Server Beheer
Deze pagina toont 1 van de servers waarvan de software pakketten worden beheerd door deze module, en de lijst met pakketten die reeds zijn geinstalleerd. U kunt de lijst uitpakken door te klikken op de folder iconen of via de details van een pakket door te klikken op de link in de lijst.

Indien u deze server wilt verwijderen van de lijst klikt u op de knop onder server details.
