Unit type

The type of systemd unit to create. The selected type controls the filename suffix and the type-specific section written to the unit file.

service creates a guided [Service] unit with command fields. timer, socket, path, and target create the corresponding unit file and show a type-specific settings field for directives in that unit's section.

All of these system units are written below /etc/systemd/system. User units are written below the selected user's ~/.config/systemd/user directory.
