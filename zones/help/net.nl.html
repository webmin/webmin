Netwerk Interface
Een zone's netwerk interface is een virtuele IP interface in het globale systeem wat is toegekend als de primaire interface voor deze zone. Ieder heeft een IP adres wat uniek moet zijn, een geassocieerde echte netwerk interface en een optionele netmask. Omdat sommige als een interface worden aangemaakt op deze pagina word het geactiveerd in de zone (er vanuit gaande dat de zone aan het werk is).
