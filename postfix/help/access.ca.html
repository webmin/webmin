Mapa d'accés de taules
El mapa opcional d'accés de taules adreça el servidor SMTP Postfix per a rebutjar o acceptar correu selectivament. L'accés es pot permetre o denegar per a noms de host, noms de domini, xarxes adreces de host o adreces de correu específics.

Mapes d'adreces de correu

  • usuari@domini coincideix amb l'adreça de correu especificada
  • domini.tld coincideix amb domini.tld com a la part del domini d'una adreça de correu
  • usuarir@ coincideix amb totes les adreces amb la part d'usuari especificada

Mapes de Noms de host/Adreces IP

  • xar.xa.adr.ess
    xar.xa.adr
    xar.xa
    net     coincideix amb l'adreça de host IPV4 o subxarxa especificada
  • xar:xa:adr:ess
    xar:xa:adr
    xar:xa
    net     coincideix amb l'adreça de host IPV6 o subxarxa especificada

Acció en coincidir

  • OK Accepta el correu.
  • REJECT [missatge] Rebutja el correu, i respon amb un [missatge] o un error genèric.
  • DISCARD [missatge] Descarta el correu en silenci; opcionalment escriu un [missatge] de registre.
  • REDIRECT spam@elteudomini.com reenvia el correu a una altra adreça
