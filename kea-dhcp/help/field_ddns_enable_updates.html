Enable updates
Controls whether this DHCP daemon sends Name Change Requests to the DHCP-DDNS daemon. This only enables the sender side; the DHCP-DDNS listener, DNS zones, and TSIG keys must also be configured for updates to succeed.

Set this to Yes when DHCPv4 or DHCPv6 should ask D2 to maintain DNS records for leases. Leave it unset to use Kea's default behavior.
