Veld Toevoegen
Op deze pagina kunt u een nieuw veld toevoegen aan de tabel. De bewerkbare parameters voor het nieuwe veld zijn :
Veld naam
De naam van dit veld in de tabel.

Type breedte (voor varchar en char velden)
Het maximum aantal karakters toegestaan voor data in dit veld.

Array veld?
Indien deze optie is ingesteld op Ja, kan het veld meerdere waardes opslaan van hetzelfde soort.

Wanneer u klaar bent klikt u op de Opslaan knop waarmee u de tabel structuur update met het nieuwe veld.
