Toegestane Hosts
Op deze pagina kunt u bepalen welke hosts toegang mogen hebben tot uw database server en wat voor soort authenticatie zij nodig hebben. Iedere toegang controle regel kan worden toegevoegd aan een enkele host, aan een geheel netwerk, aan alle netwerk hosts of alleen aan gebruikers die verbinding maken vanaf deze machine. Bovendien kan iedere regel toegevoegd worden aan een enkele geselecteerde database of aan alle databases op de server.

Wanneer u een authenticatie mode kiest voor toegestane hosts, zijn de volgende opties beschikbaar :

Platte tekst wachtwoord
De gebruiker moet een wachtwoord opgeven dat overeenkomt met het wachtwoord wat ingesteld is op de PostgreSQL Gebruikers pagina.

Versleuteld wachtwoord
De gebruiker moet een wachtwoord opgeven die versleuteld overeenkomt met het wachtwoord wat is ingesteld in PostgreSQL Gebruikers.

Geen authenticatie nodig
Er hoeft geen wachtwoord te worden opgegeven.

Weiger verbinding
Sta geen verbindingen toe vanaf deze host. Dit is handig om toegang te weigeren van een specifieke host indien een heel netwerk toegang heeft gekregen.
