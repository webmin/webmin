Webmin PostgreSQL Backup Form Help

De PostgreSQL systeem gebruiker id moet schrijf permissies hebben op iedere directory die gebruikt word als doel voor een backup.

Ongeacht de geselecteerde backup file formaat, word van iedere database een backup gemaakt in een enkele file in de vorm : <naam>.<extensie> waar <extensie> 1 van de 'post', 'tar', of 'sql' kan zijn.

Opdrachten die gespecificeerd zijn om vooraf of achteraf uitgevoerd te worden worden ook vooraf of achteraf getriggerd bij iedere database en NIET eenmalig vooraf of achteraf een gehele set databases.
