Dump niveau
Het dump niveau controleert welke zullen worden geschreven naar de backup. Indien niveau 0 is gekozen zal iedere file in de directory naar de backup worden geschreven. Anders zal alleen de files die nieuwer zijn of die gemodificeerd zijn sinds de laatste dump, of van een lager niveau worden weggeschreven.

Wanneer u een backup wilt maken van een directory wat geen filesysteem mount punt is kunt u alleen niveau 0 selecteren.
