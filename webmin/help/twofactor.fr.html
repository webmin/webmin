Authentification à deux facteurs
L'authentification à deux facteurs peut être utilisée pour ajouter une couche de sécurité supplémentaire à un compte Webmin, ainsi que le nom d'utilisateur et le mot de passe habituels.
Lorsqu'il est activé, un utilisateur devra fournir un jeton d'authentification supplémentaire qui provient généralement d'une application pour smartphone, d'un générateur OTP matériel ou d'un SMS.
Cette page vous permet de sélectionner un fournisseur d'authentification pour les utilisateurs Webmin sur votre système.
Une fois activé, chaque utilisateur peut alors s'inscrire pour deux facteurs dans le module Utilisateurs Webmin.
Les fournisseurs à deux facteurs disponibles sont:

Authy
Il s'agit d'un service commercial qui fournit une application pour smartphone et un site Web pour gérer les utilisateurs à deux facteurs.
Chaque utilisateur doit créer un compte sur le site Web d'Authy et le lier à son smartphone.

Google Authenticator
Il s'agit d'une application pour smartphone qui implémente le protocole TOTP standard.
Chaque utilisateur doit scanner un code QR à l'aide de l'application pour lier ses jetons au serveur Webmin.
